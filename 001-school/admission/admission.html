Admission Central Page

Welcome to the central hub between the Admin Dashboard and the Admissions interface. This page provides two main options:

  • Student Admission: For enrolling new students.
  • Staff Admission: For adding new staff members.

The detailed admission forms, which collect personal info, academic records, etc., are housed in their respective modules. Responsibility for each module lies with the respective department.

Before enabling these options, the system checks the Firestore document at 001-school/systemSettings to verify if the admission module is active. Only if modules.admission is true will the buttons be enabled.
